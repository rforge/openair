polar.percentile(openair)R Documentation

Polar percentile plots

Description

Shows the concentration of a pollutant in polar coordinates and information the percentiles.

Usage

polar.percentile(polar,
pollutant = "",
type = "ws",
limits = c(0, 100),
per = c(50, 95),
cols = "default",
auto.smooth = TRUE,
k = 10,
main = "",
auto.text = TRUE, ...)

Arguments

polar Mandatory. The user must supply a data frame with fields including date, pollutant, ws and wd.
pollutant Mandatory. A pollutant name corresponding to a variable in a data frame should be supplied e.g. pollutant = "nox"
type The type can be "ws" (default) "year" or "month", for plotting by wind speed ranges, year and by month of year. The choice of year is useful for examining trends and plotting by month can indicate seasonal influences.
limits ~~Describe limits here~~
per The user can provide overall percentile values to plot. This is is the form c(50, 95) e.g. to consider the 50th and 95th percentiles.
cols Colours to be used for plotting. Options include "default", "increment", "heat", "spectral" and user defined.
